Quick Install

Run these commands in your terminal (requires Node.js 18+):

npx relay-init@latest

Requires Node.js 18+ and a Cloudflare account. Or view manual setup →

Manual Setup Steps
1
Create Relay account
Done — you're logged in and your publisher account is active.
2
Install the Cloudflare Worker
Clone the worker, add your API key as a secret, and deploy to your Cloudflare account.
git clone https://github.com/hastikdan/relay-worker cd relay-worker npm install wrangler secret put RELAY_API_KEY wrangler deploy
3
Point your domain through Cloudflare
In your Cloudflare dashboard, add a Worker route for your domain. The Worker will intercept agent traffic automatically — human traffic passes through unchanged.
Route: yourdomain.com/* Worker: relay-worker
4
Verify agent traffic is flowing
Come back to the Overview tab after 24 hours. You should see your first agent requests appear in the dashboard. Most sites see traffic within hours.
